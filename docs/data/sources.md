# Data Sources — Mining KPI Tracker

## Primary Dataset
- **Name:** Open Database on Global Coal and Metal Mine Production
- **Source:** Zenodo (Vienna University of Economics and Business)
- **URL:** https://zenodo.org/records/7328050
- **Coverage:** 1,171 mines across 80 countries, 2000–2021
- **Format:** CSV
- **License:** Creative Commons Attribution

## Secondary Dataset
- **Name:** Global Mine Production by Mineral
- **Source:** Our World in Data
- **URL:** https://ourworldindata.org/grapher/global-mine-production-minerals
- **Format:** CSV
- **License:** Creative Commons Attribution

## Notes
Raw data files are not stored in this repo.
Download directly from the URLs above.
